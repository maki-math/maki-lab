1.1 Quantum Mech. for Pure States (1)

1.1.1 Hilbert Spaces

In quantum mechanics, we say the dynamics of a system is described in a Hilbert space $H$. A Hilbert space is a complex vector space with a defined inner product.

$$ \bbox[2px, border: 1px]{\text{System}\leftrightarrow\text{Hilbert Space}} $$

Vector Spaces A vector space is a set $V$ with an addition and an scalar multiplication. For arbitrary vectors $\ket{a},\ket{b},\ket{c}\in V$, and scalars $\alpha, \beta \in \mathbb{C}$, following properties must be satisfied:

  • Commutativity $\ket{a}+\ket{b}=\ket{b}+\ket{a}$;
  • Associativity $(\ket{a}+\ket{b})+\ket{c}=\ket{b}+(\ket{a}+\ket{c})$; $(\alpha\beta)\ket{a}=\alpha(\beta\ket{a})â€‹;
  • Additive identity $\exists 0\in V$, s.t. $\ket{a}+0=\ket{a}$;
  • Additive inverse
  • Distributive properties $\alpha(\ket{a}+\ket{b})=\alpha\ket{a}+\alpha\ket{b}$; $(\alpha+\beta)\ket{a}=\alpha\ket{a}+\beta\ket{a}$;
  • Multiplicative identity $1\ket{a}=\ket{a}.$

Inner Products

This is an additional operation defined in the vector space $V$. It takes two vectors to a complex number, i.e. $\left<\cdot,\cdot\right>: V\times V\rightarrow\mathbb{C}$. Here we denote $\left<\ket{a},\ket{b}\right>\equiv\braket{a|b}$. Here are some properties:

  • $\braket{a|a}=0$, iff $\ket{a}=0$;
  • $\braket{a|b}=\braket{b|a}^*$;
  • Anti-linearity $\left<\ket{a},\alpha\ket{b}\right>=\alpha\braket{a|b}$; $\left<\alpha\ket{a},\ket{b}\right>=\alpha^*\braket{a|b}$; $\left<\ket{a},\ket{b}+\ket{c}\right>=\alpha\braket{a|b}+\braket{a|c}$; $\left<\ket{a}+\ket{b},\ket{c}\right>=\alpha\braket{a|c}+\braket{b|c}$;
  • Cauchy-Schwarz inequality $|\braket{a|b}|^2\le\braket{a|a}\braket{b|b}$.

$\ket{\cdot}$ is called a ket vector, and $\bra{\cdot}$ is a bra (dual) vector in the dual space $V^*$. Therefore we have an alternative way to understand inner product: treat the dual space as a linear map $\bra{\cdot}:V\rightarrow\mathbb{C}$.

 

States of the system is described by normalized vectors (normalized: $\braket{\psi|\psi}=1$).

$$ \bbox[2px, border: 1px]{\text{States}\leftrightarrow\text{Vectors}} $$

1.1.2 Operators

When we have a system, we might want to take measurements, and we will get physical observables. They are described by operators acting on the system.

$$ \bbox[2px, border: 1px]{\text{Measurements/Observables}\leftrightarrow\text{Operators}} $$

Generally speaking, an operator turns a vector into another vector in the same vector space, i.e. $\hat{A}\ket{\psi}=\ket{\psi'}, \hat{A}: V\rightarrow V$. Physically, it turns a state into another.

Now define Hermitian conjugation (Hermitian adjoint): an operator, denoted $\hat{A}^\dagger$, is the adjoint of another operator $\hat{A}$, if it satisfy $\left<\hat{A}^\dagger\ket{a},\ket{b}\right>=\left<\ket{a},\hat{A}\ket{b}\right>$.

Adjoints properties

  • It is natural to require an adjoint has involutivity $(\hat{A}^\dagger)^\dagger=\hat{A}$;
  • Anti-linearity $(\hat{A}+\hat{B})^\dagger=\hat{A}^\dagger+\hat{B}^\dagger$; $(\alpha\hat{A})^\dagger=\alpha^*\hat{A}$;
  • Anti-distributivity $(\hat{A}\hat{B})^\dagger=\hat{B}^\dagger\hat{A}^\dagger$;

 

To make a previous state more precise, a operator that relates to a physical observable needs to be self-adjoint (or Hermitian), i.e. $\hat{A}^\dagger=\hat{A}$. Then this operator, in matrix representation, would have real eigenvalues. Suppose we have system with an initial state $\ket{\psi}$, and an operator $\hat{A}$ with a set of eigenvalue ${a_i}$, and a set of normalized either vector $\ket{a_i}$. One of the postulate of quantum mechanics states that, only eigen value can be observed, i.e. $\hat{A}\ket{a_i}=a_i\ket{a_i}$. The probability one eigenvalue is measured is given by

$$ {P(a_i)=|\braket{a_i|\psi}|^2}, $$

and state of the system after measurement collapses to $\ket{a_i}$.

Now consider a non-degenerate case, so that the number of eigenvalue equals to the dimension of the Hilbert space $d$. Using the fact that eigenvector are orthogonal to each other (and they are also normalized), i.e. $\braket{a_i|a_j}=\delta_{ij}$, we can define a projection operator

$$ \hat{P}=\sum_{i=1}^{n\le d}\ket{a_i}\bra{a_i}, $$

that is to project a vector to a subspace of the Hilbert space. When $n=d$, the projection operator gives the identity operator

$$ \hat{I}=\sum_{i=1}^{d}\ket{a_i}\bra{a_i}. $$

The operator $\hat{A}$, using the idea of spectral decomposition (or eigen decomposition), can be expressed as

$$ \hat{A}=\sum_{i=1}^{d}a_i\ket{a_i}\bra{a_i}. $$

The state $\ket{\psi}$ can also be expand in the eigen-basis

$$ \ket{\psi}=\sum_{i=0}^d\psi_i\ket{a_i}, \text{where }\psi_i=\braket{a_j|\psi}. $$
