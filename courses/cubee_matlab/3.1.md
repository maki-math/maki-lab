# 迭代法

对于线性方程组 $Ax = b$ , 如果系数矩阵$A$ 是大型稀疏矩阵(大型：矩阵的阶很高，稀疏：数值为0的元素数目远远多于非0元素的数目), 那么分解法(直接法)往往不够高效. **迭代法**生成一系列近似解. **MATLAB **提供了一些常用的迭代方法.

| 函数     | 描述               |
| :------- | ------------------ |
| pcg      | 预设共轭梯度法     |
| bicg     | 双共轭梯度法       |
| bicgstab | 双共轭梯度稳定解法 |
| cgs      | 共轭梯度平方法     |
| gmres    | 广义最小残差法     |
| lsqr     | 最小二乘法         |
| minres   | 最小残差法         |
| qmr      | 准最小残差法       |
