逻辑与控制语句

  • 选择结构

    • if, elseif, else
if expression1
        statements1
        elseif expression2
        statements2
        else
        statements3
        end

  • switch

    switch switch_expression
                case case_expression
                statements
                case case_expression
                statements
                ...
                otherwise
                statements
                end

  • 循环结构

    • for 
for index = values
        statements
        end

作为指标集的values，有以下几种形式

  • initVal:endVal 指标index将从initVal按步长为1增加至endVal，并且重复执行statements直至index大于endVal（即包含首尾）

  • initVal:step:endVal 指标index按步长step增加（或减少，当步长为负数时）至大于（或小于）endVal

     

    • while
    while expression
                statements
                end

  • break 和 continue

break 与 continue 一般需要与判断结构结合使用, 用于跳过当次循环

不同之处在于 break 会结束整个循环结构, 但continue 会进入下一次循环

例如

for i=1:5
        if i==3
        break;
        end
        disp(i)
        end
        
        1 2
    
for i=1:5
        if i==3
        continue;
        end
        disp(i)
        end
        
        1 2 4 5
